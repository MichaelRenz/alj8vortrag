Stream Operations

  • intermediate vs terminal operations - intermediate operations are producing a stream again, e.g. filter(), map(), terminal operations do something else, e.g. forEach(), count(), collect()
  • filter - build a new sequence which is the result of a filter applied to each element of the original collection
  • map - build a new sequence where each element is the result of a mapping from an element of the original sequence
  • reduce - produce a single result from all elements of the original sequence
  • collect - transforms the elements of the stream into a different kind of result, e.g. a List, Set or Map (useful builtin functions averagingInt, summarizingInt...

filter(), count() & collect() - userAge >= 18

oldsch00l 
int threshold = 18;
List<User> usersOfAge = new ArrayList<>();
int usersOfAgeCount = 0;
for(User u : userList) {
    if(u.getAge() >= threshold) {
        usersOfAge.add(u);
        usersOfAgeCount++;
    }
}
//System.out.println(usersOfAgeCount);
//System.out.println(usersOfAge.size());
vs
 count() - we simply need the number of users matching our criterion 
long usersOfAgeCount = userList.stream()
      .filter(u -> u.getAge() >= 18)
      .count();
//System.out.println(usersOfAgeCount);
vs
 collect() - we are interested in the additional info on the users matching our criterion 
int threshold = 18;
List<User> usersOfAge = userList.stream()
      .filter(u -> u.getAge() >= threshold)
      .collect(Collectors.toList());
//System.out.println(usersOfAge.size());

filter() - UserStatus.ACTIVE

List<Integer> userIDsForUsersWithFirstnameStartingWithA = userList.stream()
    .filter(u -> u.getUserStatus() == UserStatus.ACTIVE)
    .collect(Collectors.toList());

averagingInt, summarizingInt etc.

Double averageAge = userList.stream()
    .collect(Collectors.averagingInt(u -> u.getAge()));
will give us something like 43.43
IntSummaryStatistics ageSummary = userList.stream()
    .collect(Collectors.summarizingInt(u -> u.getAge()));
will give us something like IntSummaryStatistics{count=10, sum=692, min=23, average=43.43, max=96}
